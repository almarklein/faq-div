Options

Options can be set by using the data- field in the html element. * data-collapse: set to "false" to show q&a items uncollapsed. Default is "true". * data-search: set to "false" to hide the search element. Default is "true". * data-search-placeholder: custimize the placeholder text in the search input box. * data-search-info: this string is displayed when the user does a search. The text "COUNT" is replaced with the search count, and the text "SEARCH" is replaced with the current text in the search box. You can use this to customize for other languages, but also to e.g. display a mailto link.
